# ЧёПочём

Приложение для размещения объявлений с поддержкой PWA.

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

## Сборка

```bash
npm run build
```

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Запушите код:
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/obyavleniya.git
git push -u origin main
```

3. В настройках репозитория включите GitHub Pages:
   - Settings → Pages → Source: `gh-pages` branch

4. После каждого push в main ветку автоматически запустится сборка и деплой.

## Установка как PWA

После деплоя приложение можно установить на устройство:
- Откройте сайт в браузере
- В меню браузера выберите "Установить приложение"
- Или используйте кнопку "Добавить на главный экран"

## Функции

- Создание, редактирование и удаление объявлений
- Категории: Продам, Сдам, Куплю, Отдам, Обмен, Доставка, Курьер
- Чат с продавцами
- Поиск по объявлениям
- Раздел "Мои объявления" для управления своими объявлениями
- PWA поддержка для офлайн работы
